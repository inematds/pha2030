💻 O que e Claude Code

Claude Code e a interface de linha de comando oficial do Claude. Em vez de conversar pelo navegador, voce conversa com o Claude diretamente no terminal do seu computador. A diferenca fundamental: Claude Code pode ler, criar e editar arquivos, executar comandos e interagir com todo o seu sistema.

🔑 Claude.ai vs. Claude Code

  • •Claude.ai (chat): Conversa no navegador. Otimo para perguntas, brainstorming e textos curtos. Nao acessa seus arquivos.
  • •Claude Code (CLI): Conversa no terminal. Pode ler toda sua pasta de projeto, editar arquivos, rodar comandos, criar projetos inteiros. E um agente completo.
  • •Quando usar Code: Projetos com muitos arquivos, criacao de sites/apps, automacoes complexas, tarefas que exigem acesso ao sistema

💡 Dica Pratica

Voce nao precisa saber programar para usar Claude Code. Basta saber abrir o terminal, navegar ate uma pasta e digitar "claude". A partir dai, voce conversa em portugues e o Claude faz o trabalho tecnico.

2

📝 Claude.md

O arquivo CLAUDE.md e a "memoria persistente" do Claude Code. Nele, voce define quem voce e, o que espera, quais padroes seguir e o contexto do projeto. Claude le esse arquivo automaticamente toda vez que inicia uma sessao.

🔑 3 Niveis de CLAUDE.md

  • •Global (~/.claude/CLAUDE.md): Vale para todas as sessoes. Preferencias pessoais, idioma, estilo de comunicacao.
  • •Projeto (./CLAUDE.md): Vale para o projeto atual. Stack tecnologico, padroes de codigo, regras de negocio.
  • •Pasta (./pasta/CLAUDE.md): Vale para arquivos daquela pasta. Regras especificas de um modulo ou componente.

💡 Dica Pratica

Comece com um CLAUDE.md global simples: seu nome, idioma preferido (portugues), e 3-5 preferencias (ex: "sempre explique o que vai fazer antes de fazer", "use comentarios em portugues", "prefira solucoes simples"). Refine conforme usa.

3

⚡ Skills e Slash Commands

Skills sao automacoes predefinidas que voce invoca com /nome. Em vez de digitar instrucoes longas toda vez, voce cria uma skill e depois usa com um unico comando. Pense como macros inteligentes para Claude Code.

🔑 Como Funcionam

  • •Criacao: Um arquivo Markdown na pasta .claude/skills/ com instrucoes detalhadas
  • •Invocacao: Digite /nome-da-skill no Claude Code e ela executa automaticamente
  • •Exemplos: /commit (cria commit padronizado), /review (revisa codigo), /deploy (publica projeto)
  • •Personalizacao: Crie skills para suas tarefas repetitivas — relatorios, formatacao, analises

📊 Exemplos de Skills Uteis

  • /resumo - Resume todos os arquivos de uma pasta em um documento unico
  • /relatorio - Gera relatorio padronizado a partir de dados brutos
  • /traduzir - Traduz todos os arquivos .md de uma pasta para outro idioma
  • /organizar - Reorganiza arquivos em pastas por categoria automaticamente
4

🤖 Subagentes

Subagentes sao instancias de Claude que o agente principal cria para trabalhar em paralelo. E como ter uma equipe de IA: o agente principal divide o trabalho e cada subagente executa sua parte simultaneamente.

🔑 Como Funciona

  • •Divisao: Voce pede uma tarefa grande. Claude divide em subtarefas independentes.
  • •Delegacao: Cada subtarefa vai para um subagente que trabalha em paralelo.
  • •Reuniao: Os resultados sao reunidos pelo agente principal em um resultado coeso.
  • •Exemplo real: "Crie 8 paginas de modulo" — Claude cria subagentes para fazer varias paginas simultaneamente.

💡 Dica Pratica

Para aproveitar subagentes, de instrucoes claras e independentes. "Crie 5 arquivos, cada um com [especificacao diferente]" funciona melhor que "crie um arquivo e depois outro e depois outro" porque permite paralelizacao.

5

🏗️ Primeiro Projeto

Vamos criar algo funcional do zero com Claude Code. O objetivo e provar que voce pode construir coisas reais sem saber programar — usando IA como seu engenheiro pessoal.

1

Defina o Escopo

5 minutos

Escolha algo simples e util: uma pagina web pessoal, um script que organiza arquivos, uma calculadora especifica do seu trabalho, um dashboard de dados. Comece pequeno.

2

Instrua o Claude

5 minutos

Descreva exatamente o que voce quer: "Crie uma pagina HTML com [funcionalidade]. Deve ter [caracteristicas]. Use [estilo]." Quanto mais especifico, melhor o resultado.

3

Itere sobre o Resultado

15 minutos

Abra o resultado, veja o que funciona e o que nao funciona. Peca ajustes: "Mude a cor para azul", "Adicione um botao que faz X", "O grafico esta errado, use esses dados". Itere ate ficar satisfeito.

4

Publique e Compartilhe

5 minutos

Para paginas web, use GitHub Pages ou Vercel para publicar gratuitamente. Compartilhe o link com colegas. Voce acaba de criar algo funcional sem escrever uma linha de codigo.

6

🎯 Quando Usar Claude Code

Claude Code nao e para tudo. Ele brilha em cenarios especificos e saber quando usa-lo vs. quando usar o chat web maximiza sua produtividade.

✓ Use Claude Code Para

  • ✓Projetos com muitos arquivos (sites, apps, cursos)
  • ✓Tarefas que precisam ler/editar arquivos do seu PC
  • ✓Automacoes que rodam comandos do sistema
  • ✓Projetos que precisam de contexto grande e persistente

💬 Use Chat Web Para

  • •Perguntas rapidas e brainstorming
  • •Analise de imagens e documentos isolados
  • •Escrita de textos curtos sem contexto de projeto
  • •Conversas casuais e exploracoes rapidas
